Proyectos Desarrollados


Proyecto Grupal Parcial 1-Mantenimiento de Computadoras.


El proyecto consistió en desarmar, limpiar, fotografiar, documentar e identificar los componentes de una computadora previo a su reensamblado.





Proyecto Parcial (Individual) 2- Instalación de Ubuntu.


Instalar y configurar Virtual Box y Ubuntu y tomar captura de pantalla para comprobar el procedimiento. Posteriormente ejecutar una serie de comandos en la terminal.





Proyecto Parcial (Individual) - Cables de Red


Mediante cables UTP y conectores RJ45, crear un cable directo de red de Norma T568B, y uno cruzado con norma T568A.





Proyecto Parcial - Configuración de una RED LAN


Interconectar 3 computadoras mediante un switch y los cables elaborados en proyectos anteriores, configurar el IP, y habilitar el intercambio de archivos.





Proyecto Parcial (Individual) - Red LAN -- Packet Tracer


Construir una red LAN dentro de un hogar, conectando los 3 niveles en Cisco Packet Tracer.





Proyecto parcial (Individual) - Bases de Datos


Elaborar un diagrama Entidad-Relación en workbench, luego generar un script para transcribirlo en PHPMyAdmin, visualizar el Diagrama en el Diseñador de PHPMyAdmin e insertar 3 registros por tabla.

Fueron 3 ejercicios a realizar: Gestión de una Biblioteca. Control de Inventario en una Tienda. Red Social.
